# CRUD
Hospital Management CRUD in C# with MySQL
This project is a basic CRUD (Create, Read, Update, Delete) application developed in C# using MySQL as the database engine. It is designed to demonstrate fundamental database operations within the context of a hospital management system.

Overview
The application provides a simple interface to manage hospital-related data such as patients, doctors, and appointments. It serves as a practical example of how to connect a C# application to a MySQL database and perform essential operations.

Technologies Used
C# (.NET Framework/ .NET Core) – for building the application logic.

MySQL – relational database for storing hospital data.

ADO.NET / MySQL Connector – for database connectivity.


This project is intended as a learning resource for developers who want to:

Understand how CRUD operations work in C#.

Learn how to integrate MySQL with a C# application.

Explore a simple hospital management scenario as a practical example.
